# Phase 4.2 Application Drafting

Phase 4.2 generates reviewable Markdown application drafts. CareerOS still does not submit applications.

## Commands

```bash
career-os application plan <application_id|job_id>
career-os application cv-notes <application_id|job_id>
career-os application draft <application_id|job_id>
```

## Outputs

- `application-plan.md`
- `cv-notes.md`
- `application-message.md`
- `cover-letter.md`

`application draft` updates the tracker status to `drafted`.

## Rules

- Drafts must be reviewed manually.
- Do not invent experience.
- Do not overclaim missing requirements.
- Do not submit, email, or message automatically.
