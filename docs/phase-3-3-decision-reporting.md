# Phase 3.3 Decision Reporting

Phase 3.3 turns scored jobs into decision outputs.

## Tables

- `top_matches.csv`
- `high_salary_medium_fit.csv`
- `easy_wins.csv`
- `stretch_roles.csv`
- `skip.csv`
- `skill_gap_heatmap.csv`
- `red_flags.csv`
- `salary_transparency.csv`
- `remote_fit.csv`
- `best_next_actions.csv`

## Commands

```bash
career-os report
career-os show top
career-os show gaps
career-os show red-flags
career-os explain <job_id>
```

Reports render existing normalized, extracted, and scored data. They must not trigger new AI work or fetch additional remote data.
