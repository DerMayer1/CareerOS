# Phase 4.3 Interview Prep And Follow-Up

Phase 4.3 adds local interview preparation and manual follow-up tracking.

## Commands

```bash
career-os interview <application_id|job_id>
career-os applications followup <application_id> --date YYYY-MM-DD
career-os applications status <application_id> interviewing
```

## Outputs

`career-os interview` writes `interview-prep.md` in the application workspace.

The brief includes:

- role snapshot
- fit narrative
- strong talking points
- gaps to handle honestly
- likely questions
- STAR answer outlines
- questions for the interviewer
- negotiation notes
- risks to validate

CareerOS still does not send messages, schedule meetings, or apply automatically.
